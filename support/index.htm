Technical Support
Does it support Maya 2025?

Yes, the tool is built on Python 3 and is fully compatible with Maya 2020 through the latest version (2025+).

Does it require any additional plugins?

No, stbDirector operates as a standalone tool within Maya and does not require any external libraries or third-party plugins.

Licensing & Access
What is the difference between Pro and Light versions?

Light version: Designed for evaluation, giving you access to the full 1,040 camera setups for use in a single scene.

Pro version: Built for professional teams and large-scale projects. It includes multi-camera management, automated cinematic sequencing, storyboard export with technical data, and a "One-Click Batch Render" engine.

Can I use the license on multiple devices?

Standard licenses are typically for a single device. However, we offer Team bundles that support 5 or more devices depending on your production needs.

How do I receive new updates?

Updates are delivered automatically via Gumroad. You will receive an email notification whenever a new version or patch is released.
